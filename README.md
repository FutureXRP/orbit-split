# Orbit Split

A one-button arcade game: **flip orbit direction** and thread the gates.  
Easy to learn. Hard to master. Extremely “one more run”.

## Play
- Click / Tap / Space to flip orbit direction.
- Pass through the gate opening to score and build streaks.
- Near-miss the edge for bonus points (danger = profit).

## Run locally
Just open `index.html` in your browser.

## Publish on GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Source: **Deploy from a branch**
4. Branch: **main**, folder: **/root**
5. Save — your game will be live at your Pages URL.
